# 🛡️ Monitor de Integridad de Archivos (FIM)

Sistema de vigilancia activa para pequeñas emprendedoras. Detecta cambios no autorizados en sitios web y envía alertas instantáneas por Telegram.

## ✨ Funcionalidades
- **Vigilancia 24/7:** Automatizado con GitHub Actions.
- **Detección Criptográfica:** Usa SHA-256 para verificar cada bit de tus archivos.
- **Alertas Inteligentes:** Notificaciones vía Telegram API con filtros de errores de red.
- **Zero-Cost:** Diseñado para correr en infraestructura gratuita.

## 🚀 Instalación rápida
1. Clona el repo.
2. Configura tu `.env` con el TOKEN de Telegram y CHAT_ID.
3. Ejecuta `monitor.py`.
